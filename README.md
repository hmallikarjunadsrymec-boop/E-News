# E-News

A simple daily news website built with HTML, showcasing breaking news, highlights, and weather reports.

## Project Overview

E-News is a lightweight news portal that displays:
- **Breaking News**: Technology updates and learning resources
- **Today's Highlight**: Featured news content with formatted text
- **Weather Report**: Current temperature data for multiple cities
- **Read More**: Links to additional resources

## Features

- Clean, centered layout with organized sections
- HTML5 semantic structure
- Weather data presentation in preformatted text
- Linked resources (Google)
- Support for text formatting (strikethrough, size variations)

## Files

- `Enews.html` - Main HTML file for the news portal
- `ENEWS/ENEWS.html` - Additional news template
- `.vscode/launch.json` - VS Code configuration

## How to Use

1. Open `Enews.html` in your web browser
2. View the daily news, highlights, and weather information
3. Click on the provided links to access external resources

## Technologies Used

- CSS (basic)
